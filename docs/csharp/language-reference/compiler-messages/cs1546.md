## <a name="example"></a><span data-ttu-id="562ba-108">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="562ba-108">Example</span></span>

```cpp  
// CPP1546.cpp  
// compile with: /clr /LD  
using namespace System;  
public ref class MCPP  
{  
public:  
    property int Prop [int,int]  
    {  
        int get( int i, int b )  
        {  
            return i;  
        }  
    }  
};  
```
