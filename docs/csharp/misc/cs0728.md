## <a name="example"></a><span data-ttu-id="aebb5-110">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="aebb5-110">Example</span></span>  
 <span data-ttu-id="aebb5-111">El código siguiente generará la advertencia CS0728.</span><span class="sxs-lookup"><span data-stu-id="aebb5-111">The following code will generate warning CS0728.</span></span>  
  
```csharp

{  
    public void Dispose() {  }  
}

{  
    public static void dummy()  
    {  
        ValidBase vb = null;  
        using (vb)

            vb = null;  // CS0728  
        }  
        vb = null;  
    }

}  
```
