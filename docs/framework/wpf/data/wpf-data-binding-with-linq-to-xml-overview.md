### <a name="dynamic-data-binding-in-wpf"></a><span data-ttu-id="73455-136">Enlace de datos dinámicos en WPF</span><span class="sxs-lookup"><span data-stu-id="73455-136">Dynamic data binding in WPF</span></span>
