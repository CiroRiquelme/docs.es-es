## <a name="general-concepts"></a><span data-ttu-id="012be-105">Conceptos generales</span><span class="sxs-lookup"><span data-stu-id="012be-105">General Concepts</span></span>

 <span data-ttu-id="012be-109">La propiedad <xref:System.Windows.Forms.MenuStrip.MdiWindowListItem%2A> identifica el elemento de menú cuya lista desplegable se rellenará con los títulos de los elementos secundarios MDI del formulario MDI primario actual.</span><span class="sxs-lookup"><span data-stu-id="012be-109">The <xref:System.Windows.Forms.MenuStrip.MdiWindowListItem%2A> property identifies the menu item whose drop-down list you will populate with the titles of the current MDI parent form's MDI children.</span></span> <span data-ttu-id="012be-110">Por ejemplo, normalmente se enumeran los elementos secundarios MDI que están abiertos actualmente en el menú **ventana** .</span><span class="sxs-lookup"><span data-stu-id="012be-110">For example, you typically list MDI children that are currently open on the **Window** menu.</span></span>

 <span data-ttu-id="012be-112">Puede combinar elementos de menú de forma manual o automática.</span><span class="sxs-lookup"><span data-stu-id="012be-112">You can merge menu items manually or automatically.</span></span> <span data-ttu-id="012be-113">Los elementos de menú se combinan de la misma manera para ambos métodos, pero la combinación se activa de manera diferente, como se describe en las secciones "combinación manual" y "combinación automática" más adelante en este tema.</span><span class="sxs-lookup"><span data-stu-id="012be-113">The menu items merge in the same way for both methods, but the merge is activated differently, as discussed in the "Manual Merging" and "Automatic Merging" sections later in this topic.</span></span> <span data-ttu-id="012be-114">En la combinación manual y automática, cada acción de combinación afecta a la siguiente acción de fusión mediante combinación.</span><span class="sxs-lookup"><span data-stu-id="012be-114">In both manual and automatic merging, each merge action affects the next merge action.</span></span>
