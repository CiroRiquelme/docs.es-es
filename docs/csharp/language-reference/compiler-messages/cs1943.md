# <a name="compiler-error-cs1943"></a><span data-ttu-id="5fd01-102">Error del compilador CS1943</span><span class="sxs-lookup"><span data-stu-id="5fd01-102">Compiler Error CS1943</span></span>
<span data-ttu-id="5fd01-103">No se permiten expresiones de tipo 'type' en una cláusula from siguiente incluida en una expresión de consulta con el tipo de origen 'type'.</span><span class="sxs-lookup"><span data-stu-id="5fd01-103">An expression of type 'type' is not allowed in a subsequent from clause in a query expression with source type 'type'.</span></span> <span data-ttu-id="5fd01-104">No se pudo realizar la inferencia de tipos en la llamada a 'method'.</span><span class="sxs-lookup"><span data-stu-id="5fd01-104">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="5fd01-105">Todas las variables de rango deben representar tipos que se pueden consultar.</span><span class="sxs-lookup"><span data-stu-id="5fd01-105">All range variables must represent queryable types.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="5fd01-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="5fd01-106">To correct this error</span></span>  
  
1. <span data-ttu-id="5fd01-107">Asegúrese de que el tipo es un tipo consultable que implementa `IEnumerable`, `IEnumerable<T>` o una interfaz derivada, o cualquier otro tipo que tenga un modelo de consulta definido para este.</span><span class="sxs-lookup"><span data-stu-id="5fd01-107">Make sure that the type is a queryable type that implements `IEnumerable`, `IEnumerable<T>` or a derived interface, or any other type which has a query pattern defined for it.</span></span>  
  
2. <span data-ttu-id="5fd01-108">Si el tipo es un `IEnumerable` no genérico, proporcione un tipo explícito en la variable de rango.</span><span class="sxs-lookup"><span data-stu-id="5fd01-108">If the type is a non-generic `IEnumerable`, provide an explicit type on the range variable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="5fd01-109">Ejemplo</span><span class="sxs-lookup"><span data-stu-id="5fd01-109">Example</span></span>  
 <span data-ttu-id="5fd01-110">El código siguiente genera el error CS1943:</span><span class="sxs-lookup"><span data-stu-id="5fd01-110">The following code generates CS1943:</span></span>  
  
```csharp  
// cs1943.cs  
using System.Linq;  
class Test  
{  
    class TestClass  
    { }  
    static void Main()  
    {  
        int[] nums = { 0, 1, 2, 3, 4, 5 };  
        TestClass tc = new TestClass();  
  
        var x = from n in nums  
                from s in tc // CS1943  
                select n + s;  
    }  
}  
```
