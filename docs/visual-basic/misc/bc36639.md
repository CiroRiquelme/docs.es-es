# <a name="byref-parameter-parametername-cannot-be-used-in-a-lambda-expression"></a><span data-ttu-id="4ca5a-102">El parámetro 'ByRef' '\<parametername>' no se puede utilizar en una expresión lambda</span><span class="sxs-lookup"><span data-stu-id="4ca5a-102">'ByRef' parameter '\<parametername>' cannot be used in a lambda expression</span></span>
<span data-ttu-id="4ca5a-103">Una expresión lambda se declarada dentro de una declaración `Sub` o una función no puede usar cualquier parámetros `ByRef` de dicha `Sub` o función.</span><span class="sxs-lookup"><span data-stu-id="4ca5a-103">A lambda expression declared within a `Sub` or function cannot use any `ByRef` parameters of that `Sub` or function.</span></span> <span data-ttu-id="4ca5a-104">Por ejemplo, el siguiente código provocará este error porque el parámetro `ByRef` de `n` se usa en la expresión lambda.</span><span class="sxs-lookup"><span data-stu-id="4ca5a-104">For example, the following code will cause this error because the `ByRef` parameter `n` is used in the lambda expression.</span></span>  
  
```vb  
'' Not valid.
'Sub ExampleSub(ByRef n As Integer)  
  
'    Dim lambda = Function(p As Integer) p + n  
  
'End Sub  
```  
  
 <span data-ttu-id="4ca5a-105">**Identificador de error:** BC36639</span><span class="sxs-lookup"><span data-stu-id="4ca5a-105">**Error ID:** BC36639</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="4ca5a-106">Para corregir este error</span><span class="sxs-lookup"><span data-stu-id="4ca5a-106">To correct this error</span></span>  
  
- <span data-ttu-id="4ca5a-107">Asigne el parámetro de `ByRef` a una variable local y use la variable local en la expresión lambda, como se muestra en el código siguiente:</span><span class="sxs-lookup"><span data-stu-id="4ca5a-107">Assign the `ByRef` parameter to a local variable, and use the local variable in the lambda expression, as shown in the following code:</span></span>

    ```vb  
    Sub ExampleSub(ByRef n As Integer)  
  
        Dim temp = n  
        Dim lambda = Function(p As Integer) p + temp  
  
    End Sub  
    ```
